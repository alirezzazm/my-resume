# 🖼️ راهنمای جایگزینی عکس پروفایل

## مرحله 1: عکس خودت رو ذخیره کن

عکس پروفایلت (همون عکس کت و شلواری که فرستادی) رو با اسم `avatar.jpg` در پوشه‌ی `public/` قرار بده:

```
C:\Users\zm\Desktop\ppp\public\avatar.jpg
```

## مرحله 2: تموم!

سایت خودکار از این عکس استفاده می‌کنه. اگه فایل نباشه، یک avatar پیش‌فرض با حروف "AZ" نمایش داده می‌شه.

## ابعاد توصیه‌شده

- **مربعی** (مثلاً 600×600 یا 800×800)
- **فرمت**: `.jpg` یا `.png` (`.webp` هم خوبه)
- **حجم زیر 500KB** (برای لود سریع)

## محل نمایش عکس

عکس در دو جا نمایش داده می‌شه:

1. **بالای code editor در صفحه اصلی** (24×24)
2. **بخش About** (در صورت اضافه کردن)

## سایر فایل‌های ضروری

- `public/cv.pdf` — رزومه‌ی PDF تو (که الان آپلود کردی) — با کلیک روی دکمه‌ی "دانلود رزومه" بازش می‌کنه.
