# اطلاعاتی که باید بهم بدی 📝

برای اینکه پورتفولیو رو با اطلاعات واقعی خودت پر کنم، این موارد رو لطفاً برام بفرست:

## 1️⃣ اطلاعات شخصی
- نام و نام خانوادگی (فارسی + انگلیسی)
- تیتر شغلی دلخواه (مثال: «توسعه‌دهنده فول‌استک ارشد»)
- شماره تلفن
- شهر/کشور
- یک معرفی کوتاه (۲-۳ پاراگراف) به فارسی و انگلیسی

## 2️⃣ شبکه‌های اجتماعی (لینک‌های واقعی)
- GitHub
- LinkedIn
- Telegram (اختیاری)
- Twitter/X (اختیاری)
- Instagram (اختیاری)

## 3️⃣ مهارت‌ها
لیست مهارت‌هات با درصد تقریبی تسلط (۰ تا ۱۰۰). نمونه:
- Next.js — ۹۵٪
- React Native — ۹۰٪
- Docker — ۹۲٪
- ...

## 4️⃣ سوابق کاری و تحصیلی
برای هر مورد:
- نوع: کاری یا تحصیلی
- عنوان شغل/مدرک
- نام شرکت/دانشگاه
- تاریخ شروع و پایان (مثال: 2023-01)
- توضیح کوتاه (فارسی + انگلیسی)
- تکنولوژی‌های استفاده‌شده

## 5️⃣ پروژه‌ها (نمونه کارها)
برای هر پروژه:
- عنوان (فارسی + انگلیسی)
- توضیح (فارسی + انگلیسی)
- دسته‌بندی: web / fullstack / mobile / devops
- تکنولوژی‌ها
- لینک live (اختیاری)
- لینک کد (اختیاری)
- یک تصویر یا اسکرین‌شات

## 6️⃣ مقالات بلاگ (اختیاری)
اگر می‌خوای بخش بلاگ فعال باشه:
- عنوان
- خلاصه
- تاریخ
- زمان مطالعه (دقیقه)
- تگ‌ها

## 7️⃣ فایل‌های ضروری
- عکس پروفایل با فرمت `avatar.jpg` (مربعی، حداقل 400x400)
- رزومه PDF با نام `cv.pdf`

## 8️⃣ تنظیمات SMTP (برای فرم تماس)
- آدرس Gmail یا ایمیل
- App Password (برای Gmail از [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords) بساز)

## 9️⃣ اطلاعات سرور (برای دیپلوی)
- آدرس IP سرور Ubuntu
- نام دامنه (اگر داری) — مثل `alirezazamani.dev`

---

✅ **همین الان می‌تونی پروژه رو لوکال اجرا کنی** و با ویرایش `src/data/portfolio.ts` اطلاعاتت رو وارد کنی!

```bash
cd portfolio
npm install
npm run dev
```
